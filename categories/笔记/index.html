Java 动态代理

代理模式

所谓代理模式，就是指「代理对象」(或者我们形象地称呼为中介)，通过实现接口类具有了接口的能力，并通过和「实际的生产者」产生联系，将「实际生产者」的能力通过自己中转给「客户(调用者)」

客户不直接通过「实际生产者」获取信息，而是跟「代理对象(中介)」打交道获取信息
