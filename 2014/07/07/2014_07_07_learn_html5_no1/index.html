2014年7月7日

###暑假充电之——HTML5学习笔记NO.1
今天下午在慕课网上学习了HTML5的第一二两章，都是一些很基本的标签知识，下面做下笔记，以便以后复习。

  1. 首先知道了HTML5的语法编写风格和之前的都变得不一样了，变得简单易懂容易记忆。

    变化：

    doctype

    从<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN""http:www.w3c.org/TR/xhtml1/DTD/xhtml1-strict.dtd">变成了<!DOCTYPE html>
    字符编码

    从<meta http-equiv="Content=Type" Content="text/html;chatset=utf-8">变成了<meta chatset="utf-8">

    从以上可以看出HTML5的语法风格变得更加简洁，例外，HTML5可以全部使用大写，可以不使用引号，可以忽略闭合斜杠，也可以混合使用，但是为了编码的规范和代码的易懂性，还是最好不要这么做。
  2. HTML5文件基本结构
<html>
    <head>
        <title>网页的标题信息</title>
        <script>暂时还不懂，以后补充</script>
        <link>暂时还不懂，以后补充
        <style>暂时还不懂，以后补充</style>
    </head>
    <body>
        <p>段落</p>
        <hx>标题文本</hx>//x从1~6代表标题从最重要到次要
        <strong>加粗文字</strong>
        <em>斜体文字</em>
        <span>//无语义，为了设置单独的样式（比如颜色等），表示强调
        <q>引用文本</q>//无语义，表现为双引号，一般为短引用
        <blockquete>引用文本</blockquete>//长引用，表现为缩进
        <br>换行//表现为回车换行，和<img>一样无内容
        &nbsp；//表现为空格
        <!--注释文本><!-->
    </body>
</html>

以上是今天的学习内容，总结几点：

  1. 在学习过程中发现有些标签和Markdown是差不多的，因为之前写过Blog所以上手起来比较容易；
  2. Markdown是兼容HTML的，所以以上说法就能说得通了。

下面进行一点实践：

源代码如下：

<!DOCTYPE HTML>
    <html>
        <head>
            <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
            <title>Hello HTML5</title>
        </head>
        <body>
            <h1>欢迎来到HTML5的世界标题1</h1>
            <h2>欢迎来到HTML5的世界标题2</h2>
            <h3>欢迎来到HTML5的世界标题3</h3>
            <h4>欢迎来到HTML5的世界标题4</h4>
            <h5>欢迎来到HTML5的世界标题5</h5>
            <h6>欢迎来到HTML5的世界标题6</h6>
            <p>这是一个(这里有一个加了"& nbsp"的空格→)&nbsp段落</p>
            <p>这是另外一个段落</p>
            <!--这是注释><!--><br>
            <span>用蓝色字体表示强调</span><br>
            <style>span{color:blue}</style>
            <strong>定义加重语气（会加粗字体）</strong><br>
            <em>斜体</em><br>
            <q>咬定青山不放松</q>
            <blockquote>咬定青山不放松，<br>立根原在破岩中，<br>千磨万击还坚劲，<br>任尔东西南北风</blockquote>
            <img>
        </body>
    </html>
